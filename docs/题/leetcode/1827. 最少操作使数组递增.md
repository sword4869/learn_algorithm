就是 [945. 使数组唯一的最小增量.md](945. 使数组唯一的最小增量.md)  不排序的结果

```java
class Solution {
    public int minOperations(int[] nums) {
        int res = 0;
        int n = nums.length;

        for (int i = 1; i < n; i++) {
            if (nums[i] <= nums[i - 1]) {
                int add = nums[i - 1] - nums[i] + 1;
                nums[i] += add;
                res += add;
            }
        }
        return res;
    }
}
```
