# 堆

## 基本概念

**大顶堆堆顶最大，小顶堆堆顶最小**。

* 在大顶堆中，任意节点 C 与它的父节点 P 符合 $P.value \geq C.value$
* 而小顶堆中，任意节点 C 与它的父节点 P 符合 $P.value \leq C.value$

堆使用的是**完全二叉树**，特征：
* 如果从索引 0 开始存储节点数据
  * 节点 $i$ 的父节点为 $\lfloor((i-1)/2)\rfloor$
  * 节点 $i$ 的左子节点为 $2i+1$，右子节点为 $2i+2$
  * 非叶子节点范围为 $[0, size/2-1]$​

## 建堆

> 威廉姆斯算法

$n*\log n$。n次循环 * logn的插入排序

> Floyd

需要交换$2^h -h -1$次

其中 $2^h \approx n$，$h \approx \log_2{n}$，因此有时间复杂度 $O(n)$

- 大顶堆：**“从最后一个非叶子结点(size/2 - 1)往0索引，对每个节点下潜  ”**
- 小顶堆：**“从最后一个非叶子结点(size/2 - 1)往0索引，对每个节点上浮  ”**

```java
public void buildMaxHeap() {
    for (int i = size / 2 - 1; i >= 0; i--) {
        down(i);
    }
}
public void buildMinHeap() {
    for (int i = size / 2 - 1; i >= 0; i--) {
        up(i);
    }
}
```



## 下潜

1. parent自身和左右孩子left,right，选出最大值的索引为max

2. 如果max为孩子，那么

   ​	交换parent和max处的元素

   ​	以max为parent**递归**再下潜。

```java
// 递归
public void down(int parent) {
    int left = parent * 2 + 1;
    int right = left + 1;
    int max = parent;
    if (left < size && nums[left] > nums[max]) {
        max = left;
    }
    if (right < size && nums[right] > nums[max]) {
        max = right;
    }
    if (max != parent) {
        swap(max, parent);
        down(max);
    }
}
```

```java
// 循环
public void down(int parent) {
    while (true) {
        int left = parent * 2 + 1;
        int right = left + 1;
        int max = parent;
        if (left < size && nums[left] > nums[max]) {
            max = left;
        }
        if (right < size && nums[right] > nums[max]) {
            max = right;
        }
        if (max != parent) {
            swap(max, parent);
            parent = max;
        } else {
            break;
        }
    }
}
```

## 上浮

就是 `nums[left] < nums[max]` 

```java
public void up(int parent) {
    int left = parent * 2 + 1;
    int right = left + 1;
    int min = parent;
    if (left < size && nums[left] < nums[min]) {
        min = left;
    }
    if (right < size && nums[right] < nums[min]) {
        min = right;
    }
    if (min != parent) {
        swap(min, parent);
        up(min);
    }
}
```



## 添加元素

1. 添加元素的位置为arr[size+1]，它的parent = (size-1)/2
2. 上浮：添加元素从child索引开始，比较新值和parent的值，如果parent处小，则将parent赋值给child，更新child索引为parent，直到大了或 child==0 才退出循环，将child处填上新值。

## 弹出元素

交换：交换堆顶和最后一个元素

size--：表示删除元素

下潜：重新下潜/上浮堆顶

```java
public int poll(){
    int top = nums[0];
    swap(0, size - 1);
    size--;
    down(0);
    return top;
}
```



# 堆排序

1. 建大顶堆

​	从最后一个非叶子结点(size/2 - 1)往0索引，对每个节点下潜 

1. 出堆顶直到剩一个

   ​	交换：交换堆顶和最后一个元素

   ​	size--：表示删除元素

   ​	下潜：重新下潜堆顶

2. 得到升序数组。 



```java
// 912. 排序数组
class Solution {
    public int[] sortArray(int[] nums) {
        MaxHeap maxHeap = new MaxHeap(nums);
        while (maxHeap.size > 1) {
            maxHeap.poll();
        }
        return nums;
    }

}

class MaxHeap {
    public int[] nums;
    public int size;

    public MaxHeap(int[] nums) {
        this.nums = nums;
        size = nums.length;
        buildMaxHeap();
    }

    public void buildMaxHeap() {
        for (int i = size / 2 - 1; i >= 0; i--) {
            down(i);
        }
    }

    public void down(int parent) {
        int left = parent * 2 + 1;
        int right = left + 1;
        int max = parent;
        if (left < size && nums[left] > nums[max]) {
            max = left;
        }
        if (right < size && nums[right] > nums[max]) {
            max = right;
        }
        if (max != parent) {
            swap(max, parent);
            down(max);
        }
    }
    
    public int poll(){
        int top = nums[0];
        swap(0, size - 1);
        size--;
        down(0);
        return top;
    }

    public void swap(int p, int q) {
        int t = nums[p];
        nums[p] = nums[q];
        nums[q] = t;
    }
}
```
