## 递归
符合dfs 模板： [递归.md](..\..\recursion\递归.md) 

```java
// 144. 二叉树的前序遍历
class Solution {
    List<Integer> result = new ArrayList<>();

    public List<Integer> preorderTraversal(TreeNode root) {
        dfs(root);
        return result;
    }

    public void dfs(TreeNode root){
        if (root == null) {
            return;
        }
        for (int i = 0; i < 3; i++) {
            // 这里用 i 模拟
            if (i == 0)
                result.add(root.val);
            else if (i == 1)
                dfs(root.left);
            else if (i == 2)
                dfs(root.right);
        }
    }
}
```

class Solution {

## 栈

- 一个栈，先不压入root

- 循环条件：cur非空或栈不空

  当前非空，则处理**当前的左子树**：压栈、前序、更新至最左。

  当前空，则看**栈顶元素的右子树**。

  ​	没有右子树、右子树已经是pop，则弹栈更新pop；

  ​	否则，当前为右子树。

- 前序是压入cur.val，中序是压入peek.val，后序是压入更新后的pop.val。

```java
// 前、中、后
class Solution {
    public List<Integer> preorderTraversal(TreeNode root) {
        List<Integer> result = new LinkedList<>();
        
        TreeNode cur = root, pop = null;  // 当前节点, 最近一次弹栈的元素
        LinkedList<TreeNode> stack = new LinkedList<>();
        while(cur != null || !stack.isEmpty()){
            // 待处理左子树
            if(cur != null){
                stack.push(cur);
                result.add(cur.val);              // <<<<<<<< 前
                cur = cur.left;
            }
            else{// 栈顶元素的右子树
                TreeNode peek = stack.peek();
                // 没有右子树，更新pop
                if(peek.right == null){         
                    result.add(peek.val);              // <<<<<<<< 中
                    pop = stack.pop();
                    result.add(pop.val);              // <<<<<<<< 后
                }
                // 右子树处理完成，更新pop
                else if(peek.right == pop){
                    pop = stack.pop();
                    result.add(pop.val);              // <<<<<<<< 后
                }
                // 待处理右子树，更新cur
                else{
                    result.add(peek.val);              // <<<<<<<< 中
                    cur = peek.right;
                }
            }
        }
        return result;
    }
}
```

// 后序遍历
class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> res = new LinkedList<>();
        LinkedList<TreeNode> s = new LinkedList<>();
        TreeNode cur = root, pop = null;
        while (cur != null || !s.isEmpty()) {
            if (cur != null) {
                s.push(cur);
                cur = cur.left;
            } else {
                TreeNode peek = s.peek();
                if (peek.right == null || peek.right == pop) {
                    pop = s.poll();
                    res.add(pop.val);
                } else {
                    cur = peek.right;
                }
            }
        }
        return res;
    }
}
