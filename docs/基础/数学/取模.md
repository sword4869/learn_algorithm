## 正负数

绝对值相同，符号保持不变。

```java
for(int i = 0;  i < 10; i++){
    System.out.printf("%d%%3=%d,%d%%3=%d\n", i, i%3, -i, (-i)%3);
}
// 0%3=0,0%3=0
// 1%3=1,-1%3=-1
// 2%3=2,-2%3=-2
// 3%3=0,-3%3=0
// 4%3=1,-4%3=-1
// 5%3=2,-5%3=-2
// 6%3=0,-6%3=0
// 7%3=1,-7%3=-1
// 8%3=2,-8%3=-2
// 9%3=0,-9%3=0
```

## 大小

4 % 6 = 4：如果a<b，则a%b=a

6 % 4 = 2.
