## 取最右的1是第几位

lowbit ( n ) 定义为非负整数 n 在二进制表示下取最右的1是第几位。

比如5 的二进制是 ：0101 ， 所以有：lowbit ( 5 ) = 1

比如10 的二进制是 ：1010，所以有：lowbit ( 10 ) = 2

另一种角度：**当x为0时结果为0；x为奇数时，结果为1；x为偶数时，结果为x中2的最大次方的因子**

`lowbit( x ) = n & ( ~n + 1 ) = n & ( -n )`。

```java
int res = x & (~x + 1);

e.g. lowbit ( 10 ) = 2
0001010	x			2的原码
1110101	~x			-2的反码
1110110 ~x + 1		-2的补码
1000110 -x			-2的补码

  0001010
& 1110110
---------
  0000010
```
