## 索引

索引是**中间偏左**(偶数个0123是1)和**中间**(奇数个012是1)。

```java
int m = i + j >>> 1;
```

看到数组 (n -1) / 2，其实就是(0 + n - 1) / 2. 



如果想要**中间偏右和中间**，那么 + 1即可

​	中间偏右：偶数个0123是(0+3+1)/2=2

​	中间: 奇数个012是(0+2+1)/2=1

```java
int m = i + j + 1 >>> 1;
```
