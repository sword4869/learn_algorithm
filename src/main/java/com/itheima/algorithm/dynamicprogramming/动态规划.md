## 背包问题
背包问题：分数背包(贪心）、0-1背包（动态规划）、无限背包（动态规划）。

> 分数背包：贪心策略，选额单价最高。

> 0-1背包：整个物品。上一行

> 完全背包：无限物品。本行

![alt text](../../../../../../../images/image-51.png)

## 分数背包
- 按照单价逆序。
- 遍历装入背包，判断拿完和拿不完。拿完就装入，减小容量；拿不完，就装入部分，把剩余空间填满。
