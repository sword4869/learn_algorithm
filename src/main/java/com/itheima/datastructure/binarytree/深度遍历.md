深度优先遍历分先序遍历、后序遍历、中序遍历
## 递归
1. 穿入根节点
2. 递归当前节点，非空就递归子节点。
```java
class Solution {
    List<Integer> list = new LinkedList<>();
    public List<Integer> preorderTraversal(TreeNode root) {
        if(root != null){
            list.add(root.val);     
            preorderTraversal(root.left);
            preorderTraversal(root.right);
        }
        return list;
    }
}
```
## 栈

- 一个栈，先不压入root
- 循环条件：cur非空或栈不空
- 当前非空，则处理当前的左子树：压栈、前序、更新至最左。
- 当前树空，则看**栈顶元素的右子树**。没有右子树和右子树已经是pop，则弹栈更新pop；否则当前为右子树。
- 前序是压入cur.val，中序是压入peek.val，后序是压入更新后的pop.val。

```java
// 前、中、后
class Solution {
    public List<Integer> preorderTraversal(TreeNode root) {
        List<Integer> list = new LinkedList<>();
        
        TreeNode cur = root, pop = null;  // 当前节点, 最近一次弹栈的元素
        LinkedList<TreeNode> stack = new LinkedList<>();
        while(cur != null || !stack.isEmpty()){
            // 待处理左子树
            if(cur != null){
                stack.push(cur);
                list.add(cur.val);              // <<<<<<<< 前
                cur = cur.left;
            }else{
                TreeNode peek = stack.peek();
                // 没有右子树，更新pop
                if(peek.right == null){         
                    list.add(peek.val);              // <<<<<<<< 中
                    pop = stack.pop();
                    list.add(pop.val);              // <<<<<<<< 后
                }
                // 右子树处理完成，更新pop
                else if(peek.right == pop){
                    pop = stack.pop();
                    list.add(pop.val);              // <<<<<<<< 后
                }
                // 待处理右子树，更新cur
                else{
                    list.add(peek.val);              // <<<<<<<< 中
                    cur = peek.right;
                }
            }
        }
        return list;
    }
}
```

```java
// 前中
class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> res = new LinkedList<>();

        TreeNode cur = root;
        LinkedList<TreeNode> stack = new LinkedList<>();
        while(cur!= null || !stack.isEmpty()){
            if(cur != null){
                stack.push(cur);
                list.add(cur.val);              // <<<<<<<< 前
                cur = cur.left;
            }else{
                TreeNode pop = stack.pop();
                res.add(pop.val);       // <<<<<<<< 中
                cur = pop.right;
            }
        }
        return res;
    }
}
```
```java
// 后序遍历
class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> res = new LinkedList<>();
        LinkedList<TreeNode> s = new LinkedList<>();
        TreeNode cur = root, pop = null;
        while (cur != null || !s.isEmpty()) {
            if (cur != null) {
                s.push(cur);
                cur = cur.left;
            } else {
                TreeNode peek = s.peek();
                if (peek.right == null || peek.right == pop) {
                    pop = s.poll();
                    res.add(pop.val);
                } else {
                    cur = peek.right;
                }
            }
        }
        return res;
    }
}
```
