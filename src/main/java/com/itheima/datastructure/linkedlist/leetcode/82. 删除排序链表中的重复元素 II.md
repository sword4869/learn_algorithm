## pre、cur、cur.next
- 需要pre，因为连当前cur重复的元素也要删除，故而涉及到前一个结点，故而引入哨兵。
- 必须for，以防`[1, 1, 1]`删除不彻底`[1]`，再判断只剩一个而不重复。
- for分情况，重复元素结尾（不能更新cur=null，否则while中cur.next空指针还能再引入判断），重复元素非结尾。
```java
class Solution {
    public ListNode deleteDuplicates(ListNode head) {
        if(head == null || head.next == null){
            return head;
        }
        ListNode sentinel = new ListNode(-1111, head);
        ListNode pre = sentinel;
        ListNode cur = head;
        while(cur.next != null){		// 框架走 cur 和 cur.next 比较
            if(cur.val == cur.next.val){
                for(;;){
                    if(cur.next == null){   // 重复元素结尾
                        pre.next = null;
                        return sentinel.next;
                    }
                    if(cur.val != cur.next.val){    // 重复元素非结尾
                        pre.next = cur.next;
                        cur = cur.next;
                        break;
                    }
                    cur = cur.next;
                }
            }else{
                cur = cur.next;
                pre = pre.next;
            }
        }
        return sentinel.next;
    }
}
```

```java
class Solution {
    public ListNode deleteDuplicates(ListNode head) {
        if(head == null || head.next == null){
            return head;
        }
        ListNode sentinel = new ListNode(-1111, head);
        ListNode pre = sentinel;
        ListNode cur = head;
        while(cur != null && cur.next != null){ // cur != null 为了重复元素结尾的情况
            if(cur.val == cur.next.val){
                for(;;){
                    if(cur.next == null || cur.val != cur.next.val){  
                        cur = cur.next;
                        pre.next = cur;
                        break;
                    }
                    cur = cur.next;
                }
            }else{
                cur = cur.next;
                pre = pre.next;
            }
        }
        return sentinel.next;
    }
}
```

## 递归

```java
class Solution {
    public ListNode deleteDuplicates(ListNode head) {
        if(head == null || head.next == null){
            return head;
        }
        if(head.val == head.next.val){
            for(;;){
                if(head.next == null || head.val != head.next.val){
                    break;
                }
                head = head.next;
            }
            return deleteDuplicates(head.next);
        }else{
            head.next = deleteDuplicates(head.next);
            return head;
        }
    }
}
```
